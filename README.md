# ⬡ ListingLens

AI-powered Amazon listing analyzer. Paste any product URL and get a full report card —
Claude scores your title, images, bullets and copy across 6 categories, writes a
product-specific roast, and generates an AI hero image showing what your listing could look like.

**Live demo → [your-link.vercel.app](https://your-link.vercel.app)**

## Built with
- **Claude API** — analysis, scoring, roast, fix recommendations
- **Pollinations.ai** — free AI image generation, zero API key needed
- **Vanilla HTML/CSS/JS** — zero dependencies, single file, deploys anywhere

## Run it
1. Open `index.html` in any browser
2. Paste your Claude API key from [console.anthropic.com](https://console.anthropic.com)
3. Paste any Amazon product URL
4. Click Analyze

## What you get
- A → F report card across 6 categories
- Brutally honest AI roast of your listing
- AI-generated hero image showing your listing's potential
- Before / After image comparison
- Top 3 fixes ranked by revenue impact
- Competitor gap analysis
